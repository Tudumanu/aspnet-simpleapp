# Instalando o Microsoft Visual Studio Community no Windows 7

1. Busca no google.com por microsoft visual studio
2. Acesso ao link visualstudio.com, área downloads
3. Escolha do Visual Studio Community
4. Executado arquivo vs_community.exe
5. Mensagem de aviso de versão do windows antiga ou incompatível (Versão Win 7 Professional)
6. Busca por possível solução no google.com - Instalação do SP1 para Windows 7 (tempo aproximado 1h)
7. Executado novamente arquivo vs_community.exe
8. Procedendo com a instalação do Microsoft Visual Studio (tempo aproximado 3h e 30m)
9. Buscar no youtube:  asp.net hello world
10. No Visual Studio, file > new > WebSite... > Visual C# > ASP.NET Web Site
11. Clicando na seta verde (build) – Utiliza o IIS para publicar em C:\Users\<MEU_USUARIO>\Documents\Visual Studio 2015\ WebSites\
12. Uso do “refresh” dentro do Visual Studio para visualizar alterações
